ENGLISH
--------------------------------------------------------

WHAT THIS IS
A simple Windows timer. Double-click one file, wait, and
a popup reminds you when time is up. No window stays open
in the meantime.

FILES
- StartTimer.vbs   -> Double-click this to start the timer.
- Set-Timer.ps1    -> The actual logic. Edit this to change
                      the duration or the messages.
- timer-error.log  -> Only appears if something goes wrong.
                      Not present during normal use.

HOW TO USE
1. Double-click StartTimer.vbs.
2. A popup appears immediately:
      Title:   UmaReminder
      Message: 50分後にリマインダーが来ます
               ("A reminder will come in 50 minutes")
3. Close that popup. No window remains open.
4. After 50 minutes, a second popup appears automatically:
      Title:   UmaReminder
      Message: 自主トレが終わりました
               ("Self-training is finished")
5. Click OK to dismiss it. That's it.

HOW TO CHANGE THE TIME OR MESSAGE
Open Set-Timer.ps1 in Notepad and edit these three lines
near the top:

    $DefaultMinutes = 50
    $PopupTitle     = "UmaReminder"
    $PopupMessage   = "自主トレが終わりました"

Change 50 to any number of minutes you want, and edit the
text in quotes to whatever message you want. Save the file
(keep it saved as UTF-8, Notepad does this by default) and
double-click StartTimer.vbs again to use the new settings.

HOW IT WORKS (for the curious)
Double-clicking StartTimer.vbs silently launches
Set-Timer.ps1 in the background. That script:
  1. Shows the "starting" popup immediately.
  2. Registers a one-time Windows Task Scheduler task set
     to fire at (now + your chosen duration).
  3. Exits completely.
The scheduled task runs independently of this script or
any window you had open -- it is handled by Windows itself,
so it fires reliably even if you've since closed everything
or switched to a different program. When it fires, it shows
the reminder popup and then deletes itself automatically, so
no leftover scheduled tasks pile up over time.

TROUBLESHOOTING
- No popup appears at all: check the folder for a file
  named timer-error.log. If present, open it in Notepad --
  it will describe what went wrong.
- Windows SmartScreen or antivirus warning: this is normal
  for downloaded scripts. Right-click the .vbs or .ps1 file,
  choose Properties, and check "Unblock" if that option
  appears, then try again.

日本語

これは何か
シンプルなWindows用タイマーです。ファイルをダブルクリック
するだけで、指定した時間が経過するとポップアップでお知らせ
します。待っている間、画面に余計なウィンドウは残りません。

ファイル構成
- StartTimer.vbs   ・・・ ダブルクリックしてタイマーを開始
- Set-Timer.ps1    ・・・ 実際の処理本体。時間やメッセージ
                          を変更したい場合はこのファイルを
                          編集します。
- timer-error.log  ・・・ エラーが発生した場合のみ作成され
                          ます。通常の使用時には存在しません。

使い方
1. StartTimer.vbs をダブルクリックします。
2. すぐにポップアップが表示されます：
      タイトル： UmaReminder
      メッセージ： 50分後にリマインダーが来ます
3. ポップアップを閉じます。その後、画面には何も残りません。
4. 50分後、自動的に2つ目のポップアップが表示されます：
      タイトル： UmaReminder
      メッセージ： 自主トレが終わりました
5. 「OK」を押して閉じれば完了です。

時間やメッセージを変更する方法
Set-Timer.ps1 をメモ帳で開き、上部にある以下の3行を編集
してください。

    $DefaultMinutes = 50
    $PopupTitle     = "UmaReminder"
    $PopupMessage   = "自主トレが終わりました"

50 の部分を好きな分数に変更し、引用符（" "）内のメッセージ
も自由に書き換えられます。編集後はファイルを保存し（メモ帳
は標準でUTF-8として保存されます）、StartTimer.vbs を再度
ダブルクリックすれば新しい設定が反映されます。

仕組み（興味のある方向け）
StartTimer.vbs をダブルクリックすると、裏側で Set-Timer.ps1
が実行されます。このスクリプトは以下の処理を行います：
  1. 「開始しました」というポップアップをすぐに表示する
  2. 「現在時刻＋指定した時間後」に実行されるタスクを、
     Windowsのタスクスケジューラに1回限りの予約として登録
     する
  3. スクリプト自体は完全に終了する
予約されたタスクは、このスクリプトや開いていたウィンドウとは
完全に独立して動作します。Windows自体が管理しているため、
その後すべてのウィンドウを閉じたり、別の作業をしていても
確実に実行されます。実行時にはリマインダーのポップアップを
表示した後、タスク自身を自動的に削除するため、使うたびに
不要な予約タスクが溜まっていくこともありません。

トラブルシューティング
- ポップアップが全く表示されない場合：フォルダ内に
  timer-error.log というファイルがないか確認してください。
  存在する場合はメモ帳で開くと、原因が記載されています。
- Windows SmartScreen やウイルス対策ソフトの警告が出る場合：
  ダウンロードしたスクリプトではよくあることです。.vbs や
  .ps1 ファイルを右クリックして「プロパティ」を開き、
  「許可する（ブロックの解除）」という項目があればチェック
  を入れてから再度お試しください。
